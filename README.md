# studyai-landing-page
Responsive landing page built using HTML, CSS and PHP with a simple contact form.

This project is a simple responsive landing page created using HTML, CSS, and PHP.

## Features
- Clean and simple UI
- Responsive design (works on mobile and desktop)
- Navigation bar with sections (Home, Features, Plans, Contact)
- Pricing section
- Contact form using PHP
- User data stored in a text file

## Technologies Used
- HTML
- CSS
- PHP

## How it works
The contact form collects user input (name, email, message) and stores it in a text file using PHP.
